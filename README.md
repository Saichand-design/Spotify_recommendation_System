# Spotify_recommendation_System
This project builds a recommendation engine that suggests songs based on similarity and user listening patterns.

**Key Highlights**
Data preprocessing & feature normalization, 
Content-based filtering approach, 
Cosine similarity for recommendation scoring, 
Feature selection using audio attributes, 
Designed scalable recommendation pipeline.

**Recommendation Logic**
Similarity-based filtering
Audio feature comparison
Genre & mood-based clustering

**Tech Stack**
Python | Pandas | Scikit learn | Numpy | Cosine Similarity | Streamlit
